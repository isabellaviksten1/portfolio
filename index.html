App Design

We conducted user evaluations and interviews to find a solution for the problem. You can read more about it in the workbook below. AR Application

For the course Multimodal interaction and interfaces we created an AR app that lets the user launch a rocket at a virtual target that can be seen through the phone's camera. The code was written in C# and using Unity to develop the game. I focused on haptics and audio, as well as overall coding on the project together with my group members. Creators of this project: Isabella Viksten, Jackie Hellsten, Patrik Lundqvist and Casper Hildebrand.

Web Application

We created an interactive website for the course Interaction Programming and the Dynamic Web. This websiteis supposed to be used as a tool when exploring a city that you are in. You can filter what type of locations that you want to see, and when clicking on the map you will see a list with recommended places. In this project I focused on overall design and functionality of sidebar menus. Creators of this project: Isabella Viksten, Tom Nyström, Axel Svahn and Adrian Valdani.
